## 曖昧な語

``RECITATION324_110-0``の「いらっしゃる」は

- 「動詞」としての解釈（持って来るの尊敬表現）
- 「接尾辞」としての解釈（「ている」の尊敬表現）
いずれの解釈もありうる．

音声収録を聞くと，
イタコさんは動詞っぽく，
めたんちゃんとずんだもんは，接尾辞っぽい解釈をしていると推察した．
